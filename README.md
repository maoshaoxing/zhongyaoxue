# 🏥 中药学备考神器

专业高效便捷的中药学学习助手 | 纯绿色 · 无广告 · 纯公益

## 📱 手机安装（无需应用商店）

### Android 手机
1. 用浏览器打开 👉 https://maoshaoxing.github.io/zhongyaoxue/
2. 点击首页"我知道了"确认公益声明
3. 底部会弹出"添加到手机桌面"提示，点击即可
4. 如果没有弹出，手动操作：浏览器菜单 → **添加到主屏幕**

### iPhone / iPad
1. 用 **Safari** 浏览器打开上述链接
2. 点击底部中间「分享」按钮（方框+箭头）
3. 滑动找到并点击「**添加到主屏幕**」
4. 确认名称后点「添加」

> 安装后就像普通App一样使用，支持离线访问，无需联网！

## ✨ 功能

- **中药大全** — 283种药材，含归经、四气五味、功效分类，按掌握/熟悉/了解筛选
- **别名大全** — 常见中药别名对照
- **分类歌诀** — 各类中药背诵口诀
- **随机背诵** — 随机出题自测功效
- **双主题切换** — 原版蓝白/新版绿色，一键切换
- **离线可用** — PWA技术，首次加载后可离线使用

## 🚀 技术栈

纯静态网站，单文件 `index.html` + PWA Service Worker，部署在 GitHub Pages。
