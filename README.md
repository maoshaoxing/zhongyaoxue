# 中药学备考全书

基于第十版教材的中药学学习工具，支持 283 味中药的查询、分类筛选、别名对照、分类歌诀背诵和智能记忆。

## 在线访问

👉 [点击进入中药学备考全书](https://maoshaoxing.github.io/zhongyaoxue/)

## 功能特色

- **中药大全** — 按掌握/熟悉/了解筛选，查看功效、归经、四气五味。支持按类别、四气、五味多维筛选
- **别名大全** — 常见中药别名对照表，快速查询
- **分类歌诀** — 各类中药背诵口诀，助记利器
- **智能背诵** — 三种模式（药名→功效 / 功效→药名 / 归经匹配），自动记录错题
- **学习统计** — 学习进度追踪、掌握率统计、各分类掌握情况可视化
- **搜索联想** — 输入药名实时建议，快速定位

## 本地使用

直接打开 `index.html` 即可使用，无需安装任何依赖。

## 技术栈

- 纯前端 HTML + CSS + JavaScript
- PWA 支持，可离线使用
- GitHub Pages 免费托管
- Android APK 自动构建打包

## 文件结构

```
├── index.html       # 入口页面
├── style.css        # 东方本草美学设计系统
├── app.js           # 应用逻辑
├── data.js          # 中药数据（283味药+别名+歌诀）
├── manifest.json    # PWA 配置
├── sw.js            # Service Worker
├── android/         # Android APK 构建
├── icons/           # 应用图标
└── .github/         # GitHub Actions 部署配置
```
